TAME OF THONES - CSharp

************************************************************************************************
Created using Dot Net Core 3.1

To run the program, you need to download and install the Dot Net Core SDK 3.1 for your operating system from the following link: https://dotnet.microsoft.com/download/dotnet-core/3.1

After Installation:

1. Open Command Prompt

2. Browse to the folder that contains the CSProj File (TameOfThrones -> TameOfThrones)

3. Execute the following command : dotnet build -o bin

4. Execute the following command: dotnet bin/geektrust.dll Input/input1.txt 

5. Execute the following command: dotnet bin/geektrust.dll Input/input2.txt

Expected Output for Input1:
SPACE
AIR
LAND
ICE

Expected Output for Input2:
NONE
